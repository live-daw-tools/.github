## Live DAW Tools

A collection of utilities for working with Ableton's fantastic Live digital audio workstation software, including project file parsing and system configuration spelunking.

## Repos:

 - [als parser](https://github.com/live-daw-tools/als-parser) - efficient parsing of Live .als files with minimal dependencies.
 - [live-daw-info](https://github.com/live-daw-tools/live-daw-info) - system information including which Live versions are installed and each live version's plugin configuration.
